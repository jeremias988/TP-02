1.1_ Un URL (Uniform Resource Locator ) es una cadena de texto que especifica dónde se pueden encontrar algunos recursos por ejemplo (página, imagen, video, sonido).

1.2_ Una URL absoluta es una dirección web completa que incluye el protocolo, el nombre de dominio y la ruta completa del archivo o recurso

1.3_ Las URL relativas son estructuras de URL que contienen la ruta de ubicación, omitiendo el nombre del sitio web

1.4_Una dirección URL absoluta contiene toda la información necesaria para localizar un recurso. Una dirección URL relativa localiza un recurso mediante una dirección como punto de partida.

1.6_Los directorios le permiten agrupar archivos y otros directorios para organizar el sistema de archivos en una jerarquía modular, que proporciona flexibilidad y profundidad a la estructura del sistema de archivos. Un directorio​ o carpeta​ es un contenedor virtual en el que se almacenan una agrupación de archivos informáticos y otros subdirectorios, atendiendo a su contenido, a su propósito o a cualquier criterio que decida el usuario.

1.7_El directorio raíz, indicado mediante el símbolo de barra inclinada (/), define un sistema de archivos y aparece en la parte superior de un diagrama en árbol de un sistema de archivos.
